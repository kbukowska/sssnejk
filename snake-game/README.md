# Snake Web Game

A minimal, pure JavaScript implementation of the classic Snake game that runs in any modern web browser.

## Features

- **Pure JavaScript**: No frameworks or external dependencies
- **Single HTML File**: Complete game in one portable file
- **Instant Play**: No installation or setup required
- **Cross-Browser**: Works in Chrome, Firefox, Safari, and Edge
- **Offline Ready**: Runs completely offline

## How to Play

1. Open `snake-game.html` in any modern web browser
2. Use arrow keys or WASD to control the snake
3. Eat the red food to grow and increase your score
4. Avoid hitting the walls or your own body
5. Press SPACE to restart after game over

## Controls

- **Arrow Keys**: Move the snake (↑↓←→)
- **WASD Keys**: Alternative movement controls (W=up, A=left, S=down, D=right)
- **SPACE**: Restart game after game over

## Technical Specifications

- **Grid Size**: 20x20 cells
- **File Size**: Under 10KB
- **Performance**: 60fps rendering
- **Browser Support**: Modern browsers (Chrome 80+, Firefox 75+, Safari 13+, Edge 80+)

## Game Rules

- Snake starts with 3 segments
- Score increases by 1 for each food consumed
- Game ends when snake hits walls or itself
- Food spawns randomly on empty grid cells
- No time limit or speed increases
